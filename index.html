About Angular-Errorz

Angular-Errorz provides error handling for applications using AngularJS.

As its name implies, Angular-Errorz requires Angular.

Installation

In addition to downloading the source code on github, you can also install Angular-Errorz using Bower:

bower install angular-errorz
